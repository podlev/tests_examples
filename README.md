### Описание
Шаблоны doctest, pytest, unittest

### Технологии
- Python
- doctest
- pytest
- unittest

### Doctest
```
if __name__ == "__main__":
    import doctest
    doctest.testmod()
```
или 
```
python3 doctest_example -v
```
### Pytest
```
pip install pytest
```
затем
```
pytest pytest_example.py -v
```

### Unittest
 ```
unittest_example.py -v
```
